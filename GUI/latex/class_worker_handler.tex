\doxysubsection{Detailed Description}
class \mbox{\hyperlink{class_worker_handler}{Worker\+Handler}} -\/ Class to create/launch a separate thread of class \textquotesingle{}\mbox{\hyperlink{class_worker}{Worker}}\textquotesingle{} (for transferring calibration tables between the PC and a Multiple-\/\+Sensor Interface device).

{\bfseries{Description}} Class with the methods/instructions to create/launch a separate thread that will run the instructions on the class \textquotesingle{}\mbox{\hyperlink{class_worker}{Worker}}\textquotesingle{} (for transferring calibration tables between the PC and a Multiple-\/\+Sensor Interface device), thus avoiding any blocking of the main window (UI -\/ User Interface).
