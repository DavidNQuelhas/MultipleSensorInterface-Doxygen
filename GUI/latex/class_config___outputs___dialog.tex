\doxysubsection{Detailed Description}
class \mbox{\hyperlink{class_config___outputs___dialog}{Config\+\_\+\+Outputs\+\_\+\+Dialog}} -\/ Class of the dialog window used for configuring the outputs of the sensors board (OUT0, OUT1, OUT2).

{\bfseries{Description}} \mbox{\hyperlink{class_config___outputs___dialog}{Config\+\_\+\+Outputs\+\_\+\+Dialog}} is the class for the dialog used for configuring the outputs of the sensors board (OUT0, OUT1, OUT2) Each output is defined by a boolean expression ( using the logical operators \textquotesingle{}+\textquotesingle{}, \textquotesingle{}.\textquotesingle{}, \textquotesingle{}/\textquotesingle{}, and considering priorities of sub-\/expressions inside brackets \textquotesingle{}(\textquotesingle{},\textquotesingle{})\textquotesingle{} ), where the digital inputs are the variables \+: \textquotesingle{}M?\textquotesingle{}, \textquotesingle{}R?\textquotesingle{}, \textquotesingle{}C?\textquotesingle{}, \textquotesingle{}K?\textquotesingle{} . the \textquotesingle{}?\textquotesingle{} should be a character between \textquotesingle{}0\textquotesingle{} and \textquotesingle{}9\textquotesingle{} indicating what is the sensor channel number. \textquotesingle{}M?\textquotesingle{} variables are digital inputs made by using the sensor measurement, \textquotesingle{}C?\textquotesingle{} variables are digital inputs made by using the RAW value of sensor \mbox{[}in Hz or V\mbox{]}, \textquotesingle{}C\textquotesingle{} variables are digital inputs made by using the RAW counter value, \textquotesingle{}K\textquotesingle{} variables are digital inputs made by using the counter measurement .
