{\bfseries{Description}} Structure containing all data of table headers for all calibration tables to the Multiple-\/\+Sensor Interface. This includes the calibration units of all device calibration tables, and calibration units of all counter calibration functions. Also all the parameters/constants that define the calibration functions for counter measurements. Also what calibration tables are active, number of valid lines of each calib table, and what is the sensor channel mode and jumper configuration that is valid for each calib table.
