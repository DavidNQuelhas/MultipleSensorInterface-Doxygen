To be able to set/save a new board\+\_\+\+ID on a Multiple-\/\+Sensor device connected by RS485/serial, you must know the current board\+\_\+\+ID of the device and use it on the program to send the message of new board\+\_\+\+ID to that device, or alternatively use on the program the \textquotesingle{}255\textquotesingle{}(broadcast) board\+\_\+\+ID and ensure only one Multiple-\/\+Sensor device is connected on the RS485/serial cable.
