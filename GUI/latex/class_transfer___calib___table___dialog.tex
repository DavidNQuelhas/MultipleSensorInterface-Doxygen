\doxysubsection{Detailed Description}
class \mbox{\hyperlink{class_transfer___calib___table___dialog}{Transfer\+\_\+\+Calib\+\_\+\+Table\+\_\+\+Dialog}} -\/ Dialog window for showing the progress of upload/download of calibration data between PC and Multiple-\/\+Sensor device.

{\bfseries{Description}} Class of the dialog window used for showing to the user the progress of the transfer (download or upload) between a PC and a Multiple-\/\+Sensor device, of calibration tables (including the table headers) and also the calibration parameters for the counter calibration functions.
