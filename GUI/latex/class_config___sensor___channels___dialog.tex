class \mbox{\hyperlink{class_config___sensor___channels___dialog}{Config\+\_\+\+Sensor\+\_\+\+Channels\+\_\+\+Dialog}} -\/ Class of the dialog window used for configuring the trigger condition of each sensor channel used for generating its virtual digital variable

{\bfseries{Description}} \mbox{\hyperlink{class_config___sensor___channels___dialog}{Config\+\_\+\+Sensor\+\_\+\+Channels\+\_\+\+Dialog}}, the class for the dialog used for configuring the trigger/limit value of the RAW or measurement on a sensor channel that will set his corresponding value as true(1) or false(0) used as a digital input associated with each corresponding sensor channel.
