\doxysubsection{Detailed Description}
class \mbox{\hyperlink{class_calibrate___sensors___dialog}{Calibrate\+\_\+\+Sensors\+\_\+\+Dialog}} -\/ Class of the dialog window used for configuring the calibration tables of measurements on LCR/frequency/\+ADC sensor channels , and configuring the constants of calibration function of counter measurements on frequency sensor channels.

{\bfseries{Description}} \mbox{\hyperlink{class_calibrate___sensors___dialog}{Calibrate\+\_\+\+Sensors\+\_\+\+Dialog}} is the base class for the dialog used for configuring the LCR or ADC sensors calibration (applied over the RAW data) Classes that will inherit this class are \textquotesingle{}\mbox{\hyperlink{class_device___calibrate___sensors___dialog}{Device\+\_\+\+Calibrate\+\_\+\+Sensors\+\_\+\+Dialog}}\textquotesingle{} (used for configuring the device calibrations) and \textquotesingle{}\mbox{\hyperlink{class_virtual___calibrate___sensors___dialog}{Virtual\+\_\+\+Calibrate\+\_\+\+Sensors\+\_\+\+Dialog}}\textquotesingle{} (used for configuring the virtual calibrations that are only available to use on the PC, not on the sensors board) .
