Read from a Multiple-Sensor device the board_ID of that specific device/unit.

void Read_BoardID()

This function will read the board_ID number (between 0 and 255, 255 used for broadcast) from a target Multiple-Sensor device. To be able to read a board_ID from a Multiple-Sensor device connected by RS485/serial, you must know the current board_ID of the device and use it on the program to send the message of read board_ID to that device, or alternatively use on the program the '255'(broadcast) board_ID and ensure only one Multiple-Sensor device is connected on the RS485/serial cable.

Save on the Multiple-Sensor device the new board_ID of that specific device/unit.

void Send_New_BoardID()

This function will save/send a new board_ID number (between 0 and 255, 255 used for broadcast) on a target Multiple-Sensor device. To be able to set/save a new board_ID on a Multiple-Sensor device connected by RS485/serial, you must know the current board_ID of the device and use it on the program to send the message of new board_ID to that device, or alternatively use on the program the '255'(broadcast) board_ID and ensure only one Multiple-Sensor device is connected on the RS485/serial cable.
