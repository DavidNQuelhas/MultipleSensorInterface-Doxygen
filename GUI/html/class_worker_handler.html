Detailed Description

class WorkerHandler - Class to create/launch a separate thread of class 'Worker' (for transferring calibration tables between the PC and a Multiple-Sensor Interface device).

Description Class with the methods/instructions to create/launch a separate thread that will run the instructions on the class 'Worker' (for transferring calibration tables between the PC and a Multiple-Sensor Interface device), thus avoiding any blocking of the main window (UI - User Interface).
