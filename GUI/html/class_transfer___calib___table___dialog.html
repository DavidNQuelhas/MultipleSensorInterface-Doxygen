Detailed Description

class Transfer_Calib_Table_Dialog - Dialog window for showing the progress of upload/download of calibration data between PC and Multiple-Sensor device.

Description Class of the dialog window used for showing to the user the progress of the transfer (download or upload) between a PC and a Multiple-Sensor device, of calibration tables (including the table headers) and also the calibration parameters for the counter calibration functions.

Parameters
[in]*parent- The main widget/window where this dialog belongs (relevant for deleting instance of this class when no longer required).
[in]*worker_args_ptr- A variable of type 'Worker_Arguments' (structure) with all the required data(info and references (of variables) required for setting up an instance of 'Worker_Handler' for then executing the upload/download between PC and device
[in]flag_transfer_mode_bool- A boolean variable (of type 'char', '0' - false, '1' - true) used to request what direction is the data transfer, this is: '0' indicates download (PC reads the calib tables/data from the device), '1' indicates upload (PC sends new calib tables/data to the device).
